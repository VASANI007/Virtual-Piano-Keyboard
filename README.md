# 📌 Project Overview

This project is a **Virtual Piano Keyboard** built using **Python and Pygame**.

It simulates a real piano where users can press **keyboard keys or click with the mouse** to play musical notes.

The application loads sound samples for each note and plays them in real time while highlighting the corresponding keys.

Main features include:

- Interactive piano keyboard
- Real piano note sounds
- Mouse and keyboard control
- Multiple octaves
- Visual key highlighting
- Smooth audio playback

---

# 🧠 How It Works

1. The piano interface is drawn using **Pygame graphics**
2. White and black piano keys are generated dynamically
3. Each key corresponds to a **specific musical note**
4. Sound files are loaded using **pygame.mixer**
5. When a key is pressed or clicked:
   - The note sound is played
   - The key is highlighted
6. Users can change octaves using arrow keys

# ⚙ Installation

## 🪟 Windows

### 1. Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/python-virtual-piano.git
```

### 2. Go to the project folder

```bash
cd python-virtual-piano
```

### 3. Install required packages

```bash
pip install pygame
```

### 4. Run the program

```bash
python main.py
```

---

## 🐧 Linux

### 1. Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/python-virtual-piano.git
```

### 2. Go to the project folder

```bash
cd python-virtual-piano
```

### 3. Install dependencies

```bash
pip3 install pygame
```

### 4. Run program

```bash
python3 main.py
```

# 🎹 Piano Controls

| Input | Action |
|------|------|
| Mouse Click | Play piano note |
| Keyboard Keys | Play mapped piano notes |
| Left Arrow | Change right-hand octave |
| Right Arrow | Change right-hand octave |
| Up Arrow | Change left-hand octave |
| Down Arrow | Change left-hand octave |

---

# 🧑‍💻 Usage

1. Run the program
2. Click on piano keys with the mouse **or**
3. Press keyboard keys mapped to notes
4. Adjust octaves using arrow keys
5. Play melodies using your computer keyboard
